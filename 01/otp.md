This homework demonstrates the use of one time pad and the XOR operation.


The 1st test creates:
1. file.enc - the ciphertext file
2. file.key - the one time pad key
3. file.plain - the decrypted plaintext, after using the key on the cyphertext

The 2nd test:
1. Creates file.plain
2. Encrypts that plaintext file with a randomly generated key
3. Saves the generated key and cipher into files
4. Uses the key and cipher to create a new file fileorig.plain which contains the original plaintext content
